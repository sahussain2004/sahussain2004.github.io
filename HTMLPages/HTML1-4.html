Common HTML Tags

These are the most commonly used HTML and a short description of each.

Block Elements

Block elements take up their own line and take up the full width of the page.

html

This tag defines the root of an HTML Document.

body

This defines the document's body.

div

This tag defines the section of the document.

main

Specifies the main content in the document.

footer

This tag defines the footer for the document or section.

li

Defines the list item.

head

Defines the information about the document.

section

Defines a section on the document.

article

Defines the article.

p

Defines the paragraph

header

Defines the header in a document or section.

nav

Defines the navigation links.

aside

Defines the content aside from the page content.

Inline Elements

Inline elements only take up as much space on a line as they need to. They do not start new lines.

a

This tag defines a hyperlink.

img

This tag defines the image.

strong

Defines the important text.

em

Defines the emphasized text

span

Defines a section in the document.
